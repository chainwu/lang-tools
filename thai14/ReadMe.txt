EHowNet API v2.2.2
======================

EHowNet API 是為 EHowNet 的使用者所提供
的一套 API。主要提供下列幾項功能：

1. 查詢詞彙的語義訊息(E-HowNet 定義式)
2. 查詢詞彙在 E-HowNet ontology 中的 semantic type
3. 查詢兩個詞彙之間的路徑
